# BE-CyFun ID.AM-07.1 Data that the organisation stores and uses shall be identified.

**Tier:** BASIC

## Implementation guidance
The goal of this control is to ensure that all data stored and used by the organisation is clearly identified. This
supports proper data management, protection, and alignment with business and regulatory requirements.
To achieve this goal, the following should be considered:
- Data Identification
All types of data, regardless of format, location, or system, should be identified and recorded.
- Data Classification
Identified data should be classified using standard categories such as:
- Public
- Internal
- Confidential
- Restricted
- Data-to-Asset Mapping
Data should be linked to the assets that store or process it, including physical devices, systems, software,
and applications listed in the inventories from ID.AM-01 and ID.AM-02.
- Documentation and Maintenance
Data types, classifications, and associated assets should be documented and regularly updated to reflect
changes in the environment.
