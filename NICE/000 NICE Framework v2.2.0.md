# NICE Cybersecurity Workforce Framework v2.2.0

Das NICE Framework (NIST SP 800-181) beschreibt Cybersecurity-Arbeitsrollen mit zugehörigen Knowledge-, Skill- und Task-Statements (KSTs).

Jede Work-Role-Datei enthält:
- Rollenbeschreibung
- Alle KST-Statements in voller Länge
- Verknüpfte NIST CSF 2.0 Subkategorien
